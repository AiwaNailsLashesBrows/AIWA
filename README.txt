AIWA Nails Lashes Brows - Updated GitHub Pages Starter

Files included:
- index.html
- styles.css
- CNAME

How to publish:
1. Create a public GitHub repository.
2. Upload all files to the repo root.
3. Go to Settings > Pages.
4. Set Source to "Deploy from a branch".
5. Choose branch "main" and folder "/ (root)".
6. In Custom domain, enter: www.aiwanailslashesbrows.com
7. In Cloudflare DNS, create a CNAME record:
   - Name: www
   - Target: YOUR-GITHUB-USERNAME.github.io
   - Proxy status: DNS only
8. Wait for HTTPS to provision.
